FAQ
Que puis-je éditer ? L’outil FreeLogoDesign est très complet, vous pouvez modifier la couleur, la taille du texte, la police, la forme des éléments, rajouter des icônes. La seule limite c’est votre imagination.
Quels sont les formats disponibles pour mon logo ? Différents formats sont disponibles. Pour la version gratuite, le format est en PNG de 200x200 pixels (px). Pour la version haute résolution les formats sont les suivants :. PNG 200x200px, PNG 500x500px, PNG 1000x1000px, PNG 5000x5000px, JPG 5000x5000px, Fichier PDF et vectoriel SVG.
Quelles sont les utilisations que je peux faire de mon logo ? Tout dépend de votre besoin. Avec la version gratuite de votre logo, vous pouvez très aisément l’utiliser pour votre site web, votre logo Facebook ou encore pour une signature d’e-mail. Par contre, si vous souhaitez en faire un usage professionnel, comme sur une carte de visite, avoir un logo sur la fenêtre de votre magasin ou sur votre véhicule, dans ce cas, optez pour la version haute résolution.
Combien coûte un logo ? Nous offrons 3 options. Une option qui vous permet de créer vous-mêmes votre logo gratuit, une version pour un usage professionnel ( fichier haute résolution) pour $59 et enfin un service de réalisation de logos sur mesure, réalisés par nos graphistes à $199.
Dans quelle devise serais-je facturé ? Les prix sont affichés en USD
Est-ce que je peux éditer mon logo après le téléchargement ? Oui, vous pouvez modifier votre logo ( police , forme, taille , couleur ... ) et le télécharger à nouveau. Si vous avez fait choisit l’option haute résolution, vous pouvez également le modifier, mais il y a quelques restrictions comme l’ajout d’une icône ou d’un logo par exemple. Néanmoins, si vous souhaitez rajouter ces éléments, pas de problème, vous serez averti lorsque vous effectuez une action qui à besoin de procéder à nouvel achat.
Ai-je besoin de créer un compte pour utiliser FreeLogoDesign ? Non, avec Free Logo Design pas besoin de vous créer un compte.
Est-ce que je peux mettre mon logo à jour ? Oui, si vous vous rendez compte que vous êtes limité par l’usage du logo gratuit, vous pouvez en cliquant sur le lien envoyé dans l’e-mail qui contient votre logo, tout simplement choisir un fichier haute résolution.
Quels sont les droits que j'ai sur le logo ? Est-ce que j'ai le droit de mettre un copyright dessus ? Les logos sont publics, mais non exclusifs, vous pouvez donc les utiliser pour vos besoins d’entreprises de tous les jours. Avec l’éditeur Free Logo, les options de personnalisation sont nombreuses, vous pouvez donc avoir un logo qui vous ressemble et qui sort du lot. Vous pouvez mettre un copyright dessus, dans ce cas, nous vous invitons à contacter l’agence de protection de la marque qui est compétente dans votre pays afin de lancer le processus.
Est-ce que je peux vous contacter par téléphone ? Nous évitons les frais généraux coûteux, tels que les centres d’appels, pour être en mesure de toujours vous fournir une solution abordable. Notre support se fait par e-mail et nous sommes heureux de recevoir vos questions alors n’hésitez pas à nous contacter.
